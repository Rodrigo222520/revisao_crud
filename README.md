# revisao_crud

Nesse código foi feito o cadastro de usuários, tarefas, login e kanban. Ao fazer isso poderá ser visto cada tarefa dos usuários. Isso pode ser feito após registar o login e também colocar os dados do gerenciamento de tarefas. Também para ficar mais completo foi feito diagramas e modelos de casos de uso representando o cada usuário e tarefa deve ter e também os populando-os.
